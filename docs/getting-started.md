# Getting Started

> Quickstart guide for building and developing shell-chain.

## Status: STUB

## Prerequisites

- Rust (stable toolchain, latest)
- Cargo

## Build

```bash
cargo build
```

## Test

```bash
cargo test
```

## Lint

```bash
cargo clippy -- -D warnings
cargo fmt --check
```
