# shell-chain Developer Documentation

Reader-friendly documentation for understanding and contributing to `shell-chain` in its current docs-first-with-bootstrap state.

## Repository Status

The repository now contains a minimal Rust workspace bootstrap, but it does not yet provide generated API docs or a runnable node.
These documents therefore explain the intended architecture, contribution flow, and current API surface in prose first, while the early workspace continues to grow underneath them.

## Contents

| Document | Description |
|---|---|
| [Getting Started](getting-started.md) | Current onboarding path, repository status, bootstrap commands, and workspace overview |
| [API Reference](api-reference.md) | Current and planned API boundaries across the early workspace and future crates |
| [Contributing](contributing.md) | Docs-first contribution process and expectations for ongoing code and spec changes |

## Documentation Goal

A new reader should be able to understand what `shell-chain` is, what exists today, and what is still planned by reading only the files in this repository.
