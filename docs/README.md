# shell-chain Developer Documentation

Reader-friendly documentation for understanding and contributing to `shell-chain` in its current docs-first state.

## Repository Status

The repository does not yet contain a Rust workspace or generated API docs.
These documents therefore explain the intended architecture, contribution flow, and planned API surface in prose first.

## Contents

| Document | Description |
|---|---|
| [Getting Started](getting-started.md) | Current onboarding path, repository status, and planned workspace overview |
| [API Reference](api-reference.md) | Conceptual API boundaries and the public surfaces future Rust crates are expected to expose |
| [Contributing](contributing.md) | Docs-first contribution process and expectations for future code changes |

## Documentation Goal

A new reader should be able to understand what `shell-chain` is, what exists today, and what is still planned by reading only the files in this repository.
